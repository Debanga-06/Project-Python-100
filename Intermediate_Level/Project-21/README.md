# 🎵 Music Player Project

## 📌 Overview
The **Music Player Project** is a Python-based GUI application that lets users play music files stored locally.  
It uses `pygame` for audio playback and `tkinter` for the interface.  

This project is designed as a **beginner-friendly demo**, ideal for learning GUI development and multimedia handling in Python.

---

## ✨ Features
- **Play Songs** → Select a song from the playlist and play it.  
- **Stop Playback** → Stop the currently playing song.  
- **Simple GUI** → Displays the playlist and currently playing song.  
- **Automatic Folder Handling** → Uses a `songs` folder to store music files.  

---

## 💻 Supported Platforms
- **Windows**  
- **Linux**  
- **macOS**  

**Note:** `.wav` files are recommended for reliable playback; some `.mp3` files may not play depending on system codecs.

---

## 📦 Requirements
Install dependencies using pip:

```bash
pip install pygame
````

Main libraries used:

* `pygame` → Audio playback
* `tkinter` → GUI interface

---

## ▶️ Usage

1. Create a folder named `songs` in the project directory.
2. Put your music files inside the `songs` folder.
3. Run the project:

```bash
python Music_Player.py
```

4. Select a song from the playlist and click **Play** or **Stop**.
